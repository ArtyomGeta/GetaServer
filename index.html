Using example:
public class Main {
public static void main(String[] args) {
Server server = new Server();
try {
server.setPort(8070);
} catch (IllegalAccessException e) {
e.printStackTrace();
}
server.setServerListener(new ServerListener() {
@Override
public void onClientConnected(Client client) {
System.out.println("Connected: " + client.getIp());
}
@Override
public void onClientDisconnected(Client client) {
System.out.println("Disconnected: " + client.getIp());
}
@Override
public void onMessageReceived(Client client, String s) {
System.out.println("Message from: " + client.getIp() + ": " + s);
}
});
server.launch();
}
}
Server.class:
Method name Method type Method description
setPort(int) void Specifies the server port. (Default: 8080)
getPort() int Returns the server port.
getClients() List<Client> Returns an ArrayList of all connected users.
launch() void Opens the ServerSocket.
getServerSocket() ServerSocket Returns the ServerSocket.
setServerListener(ServerListener) void Specifies and override the ServerListener, that listens to "onClientConnected", "onClientDisconnected", "onMessageReceived".
stop() void Closes the ServerSocket and disconnect all connected clients.
setMaximumClientsLength(int) void Sets the maximum number of users that can connect at the same time.
getMaximumClientsLength() int Returns the maximum number of users that can connect at the same time.
Client.class:
Method name Method type Method description
print(String) void Prints the message to client.
requireAnswer(String, boolean) String Prints the message to client and wait for answer. The second parameter is responsible for call "onMessageReceived" listener after answer receiving.
disconnect() void Disconnect the client.
getSocket() Socket Returns the client Socket.
getIP() String Returns the client IP-address.
